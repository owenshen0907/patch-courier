# Code of Conduct

This project expects respectful, technical, and good-faith collaboration.

## Expected behavior

- Focus critique on code, behavior, and product decisions.
- Share context, repro steps, and evidence when reporting issues.
- Assume other contributors are working to improve the project.
- Respect the review boundaries around security-sensitive or mailbox-related code.

## Unacceptable behavior

- Harassment, personal attacks, or discrimination.
- Publishing private credentials, mailbox contents, or personal data without permission.
- Disruptive spam, bad-faith review churn, or intentionally misleading reports.

## Reporting

If you run into conduct problems, contact the maintainer through the repository profile with links, timestamps, and any relevant context.
